---
title: "Get started with the Pelupatas Backend API quickly"
description: "Learn how to authenticate with the Pelupatas API and make your first requests to manage pets and appointments in under five minutes."
---

This guide walks you through authenticating with the Pelupatas Backend API and making your first requests. By the end, you will have obtained a JWT token, listed all pet records, and created a new pet — all using `curl` from the command line.

<Note>
  All protected endpoints require an `Authorization: Bearer <token>` header. Only `POST /api/login` is public.
</Note>

<Steps>
  <Step title="Prerequisites">
    Make sure your environment meets these requirements before you start:

    - **PHP 7.4 or higher** — the API is built on PHP
    - **Apache with mod_rewrite enabled** — required for URL routing
    - **MySQL 5.7 or higher** — the database backend
    - **A REST client** — `curl`, [Postman](https://www.postman.com), or any HTTP tool you prefer
  </Step>

  <Step title="Authenticate">
    Call `POST /api/login` with your email and password to receive a JWT token.

    ```bash
    curl --request POST \
      --url http://localhost/pelupatas/backend/src/api/login \
      --header "Content-Type: application/json" \
      --data '{"email":"user@example.com","password":"yourpassword"}'
    ```

    A successful response returns a token and the authenticated user's details:

    ```json
    {
      "success": true,
      "token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...",
      "usuario": {
        "id": 1,
        "email": "user@example.com"
      }
    }
    ```

    <Tip>
      Store the `token` value in an environment variable (e.g., `TOKEN`) so you can reuse it across requests without copying it each time: `export TOKEN="eyJ0eXAi..."`.
    </Tip>
  </Step>

  <Step title="List all pets">
    Call `GET /api/mascotas` with your token in the `Authorization` header to retrieve all pet records.

    ```bash
    curl --request GET \
      --url http://localhost/pelupatas/backend/src/api/mascotas \
      --header "Authorization: Bearer $TOKEN"
    ```

    The API returns a JSON array of all active pet records stored in the database.
  </Step>

  <Step title="Create a pet">
    Call `POST /api/mascotas` to add a new pet record. Provide all required fields in the request body.

    ```bash
    curl --request POST \
      --url http://localhost/pelupatas/backend/src/api/mascotas \
      --header "Authorization: Bearer $TOKEN" \
      --header "Content-Type: application/json" \
      --data '{
        "nombre": "Thor",
        "raza": "border collie",
        "edad": 5,
        "observaciones": "Pastor, fiel",
        "id_dueno": 2
      }'
    ```

    On success, the API returns `{"success": true}`.
  </Step>
</Steps>
