Pelupatas Backend API powers a pet grooming (peluquería canina) management system. It exposes RESTful endpoints for handling pets (mascotas), appointments (citas), and user authentication — all secured with JWT tokens and backed by a MySQL database.

## How it works

<Steps>
  <Step title="Authenticate">
    Call `POST /api/login` with an email and password. You'll receive a JWT token valid for 1 hour.
  </Step>
  <Step title="Include your token">
    Add `Authorization: Bearer <token>` to every request for protected endpoints (mascotas, citas).
  </Step>
  <Step title="Call the API">
    Use the RESTful endpoints to manage pets and appointments. All responses are JSON.
  </Step>
</Steps>

<Note>
  All endpoints are served from the base URL `http://localhost/pelupatas/backend/src/api`. See the [API Overview](/api-reference/overview) for full details.
</Note>
