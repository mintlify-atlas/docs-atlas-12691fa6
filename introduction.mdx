Pelupatas Backend API is a PHP REST API designed for pet grooming (peluquería canina) salon management. It gives you full control over pet records (mascotas), grooming appointments (citas), and user authentication — all through a clean RESTful interface secured with JWT Bearer tokens and backed by a MySQL database.

## Key features

- **JWT authentication** — All protected endpoints require a Bearer token obtained via `POST /api/login`. Tokens are signed with HS256 and expire after 1 hour.
- **Soft-delete for pets** — Pet records are deactivated rather than permanently removed, preserving data integrity.
- **CORS support** — The API sets permissive CORS headers (`Access-Control-Allow-Origin: *`) so you can call it from any frontend origin.
- **MySQL with PDO** — All database access goes through PHP Data Objects (PDO), providing consistent and safe query handling.

## Base URL

All endpoints are served from:

```
http://localhost/pelupatas/backend/src/api
```

## Available resources

| Resource   | Path            | Auth required |
| ---------- | --------------- | ------------- |
| Login      | `/api/login`    | No            |
| Pets       | `/api/mascotas` | Yes           |
| Appointments | `/api/citas`  | Yes           |
